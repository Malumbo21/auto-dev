AutoDev Xiuper - AI-native development platform built on Kotlin Multiplatform

One Platform. All Phases. Every Device.

AutoDev Xiuper covers all 7 phases of SDLC (Requirements, Development, Review, Testing, Data, Deployment, Operations) and supports 8+ platforms (IDEA, VSCode, CLI, Web, Desktop, Android, iOS, Server).

Key Features

  • One Platform: Unified Kotlin Multiplatform architecture - write once, deploy everywhere
  • All Phases: 7 specialized agents covering complete software development lifecycle
  • Every Device: Native support for 8+ platforms with zero compromise on performance

Agents

  • Coding Agent: AI-powered code generation and completion
  • Review Agent: Automated code review with AI insights
  • Document Agent: Intelligent documentation generation
  • Domain Dict Agent: Domain-specific terminology management
  • Artifact Agent: Reversible artifact generation with live preview
